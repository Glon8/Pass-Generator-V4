Password Generator v4

Info

  • REPEATABLE - Blocks characters from repeating.
  • SYMBOLS - Enables to use symbols in the password.
  • LETTERS - Enables to use letters in the password.
  • CAPITALS - Enables to use capitals in the password.
  • Password length by deffault is 8, at max 60 and at minimum 4.

Tip: Add more numbers as you add new type of symbols in. (symbols, letters, capitals)
